# Next Gen Robo Car or Dog

An AI-powered robotics platform designed for autonomous navigation, obstacle avoidance, object detection, and intelligent interaction. The project combines Computer Vision, Artificial Intelligence, IoT, and Embedded Systems to create a versatile robotic vehicle or quadruped robot capable of operating in real-world environments.

## Features

* Real-time object detection using YOLOv8
* Autonomous navigation and path planning
* Obstacle detection and avoidance
* Live camera streaming
* ESP32 and Raspberry Pi integration
* Remote monitoring and control
* Voice command support
* Environmental sensing
* IoT connectivity
* Modular robotic architecture
